# Simão Rodrigues – Portefólio

Portefólio pessoal desenvolvido com **HTML5, CSS3, JavaScript** e **Bootstrap 5**.

Este projeto apresenta informação pessoal, competências técnicas, projetos académicos/pessoais e formulário de contacto.

---

## Demonstração

Site publicado em GitHub Pages:

```text
https://simaofrodrigues.github.io/
```

---

## Tecnologias usadas

- **HTML5** – estrutura semântica das páginas.
- **CSS3** – design visual, responsividade, animações e componentes personalizados.
- **JavaScript** – tradução PT/EN, comportamento da navbar, botão de voltar ao topo e animações por scroll.
- **Bootstrap 5** – grelha responsiva, navbar, offcanvas e componentes base.
- **Bootstrap Icons** – ícones do site.
- **Formspree** – envio do formulário de contacto.

---

## Funcionalidades

- Design moderno com efeito visual tipo glassmorphism.
- Layout responsivo para desktop, tablet e telemóvel.
- Menu mobile com Bootstrap Offcanvas.
- Secções principais: Início, Sobre, Competências, Projetos e Contacto.
- Sistema de idioma PT/EN com `localStorage`.
- Grelha de projetos com cartões visuais.
- Animações suaves ao fazer scroll.
- Botão de voltar ao topo.
- Formulário de contacto integrado com Formspree.
- Melhorias de acessibilidade, incluindo link para saltar para o conteúdo principal.

## Como correr localmente

Abrir diretamente o ficheiro `index.html` no browser ou usar um servidor local:

```bash
python -m http.server 5500
```

Depois abrir no browser:

```text
http://localhost:5500
```
